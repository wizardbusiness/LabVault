### VM Operations

```ad-bash
	
	
	# VM operations (same pattern)
	qm create    # Create new VM
	qm start     # Start VM
	qm stop      # Stop VM
	qm set       # Modify VM config
	qm terminal  # Get console access
```
### Container Operations

```ad-bash
	# Container operations
	pct create   # Create new container
	pct start    # Start container
	pct stop     # Stop container
	pct set      # Modify container config
	pct enter    # Get shell inside container
	pct exec 
```
