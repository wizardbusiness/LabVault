1. In the Web UI for the client to which you wish to add a device, click the ID value next to the 'Identification' property under 'This Device'![[image-37.png]]
2. On the device you wish to add, click 'Add Device'
3. 'Scan using camera' and hold the device so that the QR code appears in the viewfinder. The device ID should be entered automatically once the QR code is scanned. Click 'Add'
4. The device should now appear under 'Associated Devices'
5. Back in the Web UI you should now see a dialog telling you that new device wants to connect. Click 'Add Device'![[image-38.png]]
6. In the Add Device interface either accept the device name or choose another one (The device will appear with this name within the client Web UI). ![[image-41.png]]
7. Optional: In the Sharing tab configure shared folders and device sharing settings ![[image-40.png]]
8. Optional: In the Advanced tab configure syncing Rate Limits, device share password protection, compression, and connection management ![[image-39.png]]
9. Click 'Save'
