1.  In the Syncthing Client Web UI, Click 'Add New Folder'![[image-34.png]]
2. In the Add Folder UI  ![[image-35.png]]
	1. Optional: Choose a Folder Label - will show up instead of the folder ID on remote devices, good if you have multiple connected folders
	2. Leave the Folder ID as-is, or choose a new unique alphanumeric identifier
	3. Choose the Folder path to sync. Note: This is from the mount point, so something like path/to/mount/folder_to_sync will just be /folder_to_sync
	4. Click 'Save'
3. Success! The folder now shows up in the syncthing Web UI with status *Unshared*
