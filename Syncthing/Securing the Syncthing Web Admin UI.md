1. After setting up syncthing for the first time, navigate to the Web UI Url and Port. If it is running you will see the Syncthing Admin Interface
	![[image-26.png]]
	
2. Inside the warning advising that the Admin interface can be accessed without a password, click 'Settings' and go to the 'GUI' tab. 
	![[image-25.png]]
	
3. Set the GUI authentication Username and Password
	![[image-29.png]]

4. Set the GUI to use HTTPS - Otherwise credentials will be sent unencrypted over the network. Not a great idea even if you're only accessing the server from your local domain. Note: If you are ONLY accessing the web ui from localhost (same machine) it is okay to leave this unchecked
5. Optional: Set GUI Theme to Dark. 
6. Click 'Save'. The web ui login screen will immediately appear. 
7. Enter your new credentials and login
	![[image-31.png]]
8. The authentication warnings should now be gone. 
	![[image-33.png]]
