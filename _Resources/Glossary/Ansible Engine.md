##### Definition

**`Ansible Engine`**

**Old (pre-2021) name for the core automation runtime which was sold with enterprise support and bundled with a curated set of modules. Ansible Core (pre-2021). Split into Ansible Core (the free/open-source runtime), and Ansible Automation Platform (enterprise product)  **
