##### Definition

**`Command (PowerShell)`**

**A named instruction entered into the [[Powershell#Definition|PowerShell ]]shell that tells the operating system or an application to perform a specific action.  Powershell commands take the form of *cmdlets* (e.g., `Get-Item`, `Set-Location`), *aliases*, *functions*, or *external executables*, and can be chained together using the *pipeline* (`|`) to pass output from one command as input to the next.**
