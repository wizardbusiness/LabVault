##### Definition

**`Command Line`**

**A text-based interface where a user types instructions directly to the operating system, allowing control over files, processes, and programs without a graphical interface — commands are entered at a *prompt* and executed by a *shell* such as Bash or Zsh.**

###### **Synonyms**

- CLI (Command Line Interface)
- Terminal
- Console
