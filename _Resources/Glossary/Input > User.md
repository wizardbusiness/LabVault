##### Definition

**`User Input`**

**Any data or signal a person sends to a computer system.**
###### Expand Detail

> [!detail]- ***+***
> **Includes actions such as keystrokes, clicks, touch, or voice — which the system receives, processes, and responds to, forming one half of the _input/output (I/O)_ cycle that drives all human-computer interaction.**
> 

> [!detail]- *Synonyms*
> - Input
> - User Action
> - User Event
