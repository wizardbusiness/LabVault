##### Definition

**`Kernel`**

**The core program of an operating system that runs continuously in privileged _kernel space_, acting as a bridge between hardware and software by managing the CPU, memory, storage, and devices — all other programs run in _user space_ and must request hardware access from the kernel via _system calls_.**

###### **Synonyms**

- OS Kernel
- System Kernel
