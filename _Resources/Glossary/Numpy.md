##### Definition

**`numpy`**

*A Python library that provides fast, efficient tools for working with arrays and mathematical operations on large datasets.*
##### ---
> [!detail]- More about `numpy`
> 
> **NumPy (Numerical Python) enables you to create multi-dimensional arrays and perform vectorized calculations on them, meaning you can run mathematical operations across entire datasets at once rather than looping through individual elements, making it dramatically faster than pure Python. It's foundational for scientific computing, data analysis, and machine learning workflows.**
