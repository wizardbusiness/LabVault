##### Definition

**`Operating System`**

**The master software layer that manages a computer's hardware and provides a stable platform for all other software to run on.**

###### Expand Detail

> [!detail]- ***+*** 
> **Sits between the raw hardware (CPU, RAM, storage, I/O devices) and the user-facing applications — handling _process scheduling_, _memory allocation_, _file system_ management, and _device communication_ via _drivers_. Without an OS, each program would need to directly control hardware itself. Common examples include Windows, macOS, Linux, and iOS.**

> [!detail]- _Synonyms_
> 
> - OS
> - Platform
> - System Software
