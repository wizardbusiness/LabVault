##### Definition

**`PowerShell`**

**A [[Command Line#Definition|command-line]] [[Shell#Definition|shell]] and scripting language built by Microsoft that lets users automate system administration tasks by running _cmdlets_, manipulating _objects_, through a _pipeline_, and interacting directly with Windows internals like the _registry_, _file system_, _Active Directory_, and the _.NET_ framework — available on Windows by default and cross-platform as _PowerShell Core_.**
###### **Synonyms**

- Windows PowerShell _(the original Windows-only version)_
