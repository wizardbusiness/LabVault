##### Definition

**`Shell`**

**A [[Program#Definition|program]] that translates between [[Input > User#Definition|user inputs]] and the operating system *[[Kernel#Definition|kernel]]* — it reads commands typed at the *[[Command Line#Definition|command line]]* (or written in a *script*), translates them into system calls, and returns output; multiple shells exist (such as *Bash*, *Zsh*, and *Fish*), each with its own syntax and features, but all serving as the outermost interactive layer through which a user drives the OS.**

###### **Synonyms**

- Command Interpreter
