##### Definition

**`Subcommand`**

**A secondary keyword passed immediately after a main CLI command that selects a specific _action_ or _mode_ within that command's namespace — for example, in `git commit`, `commit` is the subcommand that tells the `git` program _which operation_ to perform, as opposed to `git push` or `git pull`.**

###### **Synonyms**

- Verb (in CLI design terminology)
